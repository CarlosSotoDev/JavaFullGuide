Tipos de Variables

En esta sección, aprenderás sobre los diferentes tipos de variables en Java, incluyendo int, float, double, char, boolean, y más.


public class Variables {
    public static void main(String[] args) {
        int numero = 10;
        float decimal = 3.14f;
        double grande = 3.14159;
        char letra = 'A';
        boolean verdad = true;
        System.out.println("Número: " + numero);
        System.out.println("Decimal: " + decimal);
        System.out.println("Grande: " + grande);
        System.out.println("Letra: " + letra);
        System.out.println("Verdad: " + verdad);
    }
}
